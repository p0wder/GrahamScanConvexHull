Graham scan Algorithm for solving the convex hull challenge
contains 3 files: MyLine.java, MyPoint.java, TheHull.java(main)

This program uses a .txt file for input.  The format of input
is as follows (point, x, y):
a, 1, 2
b, 3, 4
c, 5, 6
etc...

Enjoy!
